-Seguimiento de actividades-

*En este apartado se presenta un sitio web que podrá ser utilizado para el seguimiento de tareas del usuario, cuenta con un formato simple y sencillo para facilitar al usuario el control de sus deberes.*

-Lenguajes utilizados-

•*JavaScript*

•*Css*

•*React*

•*Github*

-Link del sitio web-

https://seguimiento-de-actividades-112c.vercel.app/

-Dinámica del sitio web-

•*Introducir la actividad o tarea.*

•*Una vez que la tarea este completa hacer click en la tilde y automáticamente aparecerá en tareas completas.*

•*En caso de no querer seguir teniendo cierta actividad, se podrá eliminar dicha actividad haciendo click en el icono del tacho de basura.*

•*Se puede filtrar para ver actividades completas, incompletas o ambas a la vez.*
